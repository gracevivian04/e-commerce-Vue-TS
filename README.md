# online-shop

## Installation
```
$ git clone https://github.com/gracevivian04/e-commerce-Vue-TS.git
$ cd online-shop
$ npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
